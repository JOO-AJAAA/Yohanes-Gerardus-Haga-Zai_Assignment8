# BEST - Belajar CSS HTML

Aplikasi web sederhana untuk builder web dengan sistem routing SPA (Single Page Application).


## ✨ Fitur

- **Routing Dinamis**: Navigasi antar halaman tanpa reload halaman
- **Responsive Design**: Desain yang responsive dengan footer fixed
- **Form Pemesanan**: Form lengkap untuk pemesanan pembuatan website
- **Data Profil**: Tabel data diri dengan kontak person (WhatsApp & Email)
- **Map Lokasi**: Embedded Google Maps untuk menunjukkan lokasi bisnis

## 📄 Halaman yang Tersedia

### 1. Home (`/`)

- Deskripsi singkat tentang BEST
- **Form Pemesanan Web** dengan field:
  - Nama Lengkap
  - Email
  - Nomor Telepon
  - Jenis Website (dropdown)
  - Deskripsi Singkat

### 2. About (`/about`)

- Informasi tentang website
- **Tabel Data Diri** berisi:
  - Nama
  - Tempat Lahir
  - Tanggal Lahir
  - Alamat
  - Pekerjaan
  - Kontak Person (WhatsApp & Email)

### 3. Map (`/map`)

- **Embedded Google Maps** menampilkan lokasi bisnis
- Dapat di-zoom dan di-explore secara interaktif

## 🚀 Cara Menggunakan

1. Buka file `index.html` di browser
2. Gunakan navbar (Home, About, Map) untuk navigasi
3. Isi form pemesanan di halaman Home jika perlu
4. Lihat data profil di halaman About
5. Cek lokasi di halaman Map

## 🛠 Teknologi

- **HTML5**: Struktur halaman
- **CSS3**: Styling dan layout
- **JavaScript (Vanilla)**: Routing dan interaksi

## 📌 Catatan

- Form belum terhubung ke backend (untuk submit data belum ada server)
- Data di tabel About adalah dummy data, bisa disesuaikan
- Footer selalu berada di paling bawah dengan posisi fixed
- Routing menggunakan History API untuk navigasi tanpa reload
